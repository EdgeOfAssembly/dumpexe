ICON - QUEST FOR THE RING
-------------------------

This game requires FCBS=10 in CONFIG.SYS. Normally this value is 4 and causes
"FCB not available" error when restoring saved position and the game will
delete such file saying it's corrupted. Additionaly you may need to run
it in dos-box as DOS 7.x doesn't seem to emulate FCB access properly in
pure dos.

Some gfx may seem to be a bit garbled because the game uses hybrid text mode
to draw the gfx. The 8x8 character set (fonts) is a bit different than the one
used on CGA cards.
